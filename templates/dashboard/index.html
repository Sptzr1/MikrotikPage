Bienvenido al Dashboard Mikrotik

¿Qué es esta aplicación?

Esta aplicación web te permite administrar de forma centralizada tu dispositivo MikroTik con RouterOS. Desde aquí, puedes:

  • Gestionar usuarios Hotspot, incluyendo creación, edición y eliminación.
  • Monitorear y configurar interfaces de red.
  • Controlar enlaces activos, desconectar usuarios y gestionar pagos.
  • Administrar colas (Queues) para controlar el ancho de banda.
  • Supervisar conexiones PPP activas.

Usa la barra lateral para navegar entre las diferentes secciones y comenzar a gestionar tu red de manera eficiente.
